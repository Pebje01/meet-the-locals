# Meet the Locals

Persoonlijke reisblog — meetthelocals.nl

## Tech Stack

- **Framework:** Next.js 15 (App Router, RSC)
- **CMS:** Payload CMS 3.x (headless, als Next.js plugin)
- **Database:** PostgreSQL 16
- **Styling:** Tailwind CSS 4
- **Animaties:** Framer Motion
- **Kaarten:** Mapbox GL JS
- **AI:** Anthropic Claude API via Payload hooks
- **Hosting:** Hetzner CX23 + Coolify (Docker)
- **Analytics:** Plausible / Umami

## Development

```bash
pnpm install
pnpm dev
```

## Deployment

Git push → Coolify auto-deploy
